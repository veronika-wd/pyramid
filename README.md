# Финансовая пирамида МММ

Необходимо создать проект для финансовой пирамиды.\
Изначально есть один юзер, он находится в верхушке пирамиды.

### Механика следующая:
* Зарегистрироваться в системе можно только по чьей-то реферальной ссылке.
* Регистрируясь, реферал встает в структуру под владельца реферальной ссылки (надстоящего 1‑го уровня).
* После регистрации новый участник не может приглашать рефералов, пока не купит слоты.
* Покупка одного слота дает возможность пригласить одного реферала.
* Если у участника все слоты заполнены, то реферал не может зарегаться.

### Покупка слота — основная цель существования пирамиды.
Когда участник тратит деньги, они распределяются следующим образом:
* Надстоящий 1‑го уровня получает 40% от суммы.
* Надстоящий 2‑го уровня получает 30% от суммы.
* Надстоящий 3‑го уровня получает 10% от суммы.
* Остальное уходит владельцу пирамиды.

### Механика работы с финансами:
* Есть форма пополнения баланса: пишешь сумму, жмешь пополнить и получаешь на баланс сумму.
* Все манипуляции с деньгами логируются.
* Участник может посмотреть своих непосредственных рефералов 1‑го уровня, 2-го уровня, 3-го уровня. Детально: у какого реферала сколько его рефералов.
* Участник может посмотреть лог движения финансов: расходы, пополнения.
* Участник может видеть свой баланс (а зачем еще ему заходить на этот сайт)
* Участник может скопировать свою реферальную ссылку
